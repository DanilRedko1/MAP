# Angular 13 ArcGIS 4.23 Starter

This project is a minimal Angular 13 application that embeds an ArcGIS JavaScript 4.23 map using the `@arcgis/core` package.

## Features

- Angular 13 application structure
- ArcGIS 4.23 `MapView`
- Search widget
- Scale bar
- Graphics layer with sample locations and popups

## Prerequisites

This workspace currently does not have Node.js/npm installed, so dependencies could not be installed here.

Install a Node.js version compatible with Angular 13, then run:

```bash
npm install
npm start
```

The app will be served at `http://localhost:4200/`.
